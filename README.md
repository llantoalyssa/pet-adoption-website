# Dog Adoption Center Website

This is a web-based Dog Adoption System that allows users to browse dogs, submit adoption requests, track their adoption status, and subscribe to a weekly newsletter. Admins can manage dogs and adoption requests.

---

## Website Access

- Homepage:  
  `http://localhost/pet-adoption-website/index.php`

- Dogs Listing:  
  `http://localhost/pet-adoption-website/dogs.php`

- My Adoption Requests (User must be logged in):  
  `http://localhost/pet-adoption-website/my-adoptions.php`

---

## User Features

1. Register and Login (Email & Google SSO)
2. Email verification required before adoption
3. View available dogs
4. Submit adoption requests
5. View adoption status (Pending / Approved / Rejected)
6. Cancel pending adoption requests
7. Subscribe to the weekly newsletter (email confirmation sent)
8. Chat support via Tidio plugin
9. Social media share buttons
10. Image map integration
